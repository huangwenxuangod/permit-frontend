# 前端调用流程（V1）

- 规格默认锁定：用户在规格入口点击后，规格即确定为 `selectedSpecCode`
- 选图：拍摄指引页选择图片，写入 `selectedImagePath`
- 检测页：
  - 上传原图：`POST /api/upload` → `objectKey`
  - 创建任务：`POST /api/tasks` → `taskId`、`baselineUrl`、`processedUrls.white`（或轮询 `GET /api/tasks/{id}`）
- 预览页：
  - 仅前端切换底色，记录 `previewColor`，展示 `processedUrls[color]` 或 `baselineUrl`
- 订单确认：
  - 绑定最终色 `finalColor`，创建订单 `POST /api/orders`（支付另行对接）
- 支付成功后生成：
  - 按需生成底色：`POST /api/tasks/{id}/background`
  - 如需排版照：`POST /api/tasks/{id}/layout`
  - 下载信息：`GET /api/download/{taskId}`
